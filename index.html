Сортировка пузырьком

+Как пользоваться сайтом?

  1. Выберите количество элементов массива. По умолчанию - 20. Минимальное значение - 2. Максимально - 500
  2. Нажмите кнопку "Создать массив". Если массив подходит, можно приступать к сортировке и нажимать кнопку "Сортировать массив" или сгенерировать его повторно.
  3. Так же можно ввести массив руками через запятую в формате 111, 111. Допустимые для ввода числа находятся в диапазоне от -999999 до 999999. Для этого нужно убрать чекбокс "Автоматическая генерация массива" Максимальное количество элементов - 500.
  4. Кнопки "Старт", "Продолжить" и "Отменить сортировку", сответственно приостанавливают, продолжают и отменяют процесс сортировки.
  5. После сортировки можно создать новый массив и повторить сортировку.
